# UI Design Using Component Libraries
by Tanmoy Talukdar.

This project demonstrates user interface design using **Bootstrap** and **Material UI** in a **React Single Page Application**.

All four experiments are implemented in a single React component.

---

## Experiments Implemented

1. **Bootstrap Form UI**  
   - Responsive form using Bootstrap components.

2. **Card-Based Layout**  
   - Grid system with multiple cards.

3. **Material UI Form**  
   - Form built using Material UI components.

4. **Responsive Navigation Bar**  
   - Navbar created using Bootstrap.

---

## Technologies Used

- React
- Bootstrap
- Material UI (@mui/material)
- Node.js
- npm
